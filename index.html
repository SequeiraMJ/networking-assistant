Networking Assistant

Networking Assistant is a tool designed to help you in most repetitive calculations for network professionals. It aims to make VLSM, FLSM, Binary ANDing and Numerical Base conversion as easy as possible. You only need to type in information required as needed and Networking Assistant will perform those calculations for you. You are just a couple of clicks away from your results. I hope you enjoy this tool as much as I did building it.

  • VLSM

    Variable Length Subnet Mask

  • FLSM

    Fixed Length Subnet Mask

  • ANDing

    Binary ANDing calculations

  • Bases

    Binary, octal, decimal and hexadecimal
